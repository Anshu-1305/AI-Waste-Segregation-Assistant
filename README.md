## ğŸ¤– AI Classification System
The system classifies waste into four main categories:

### 1. **Wet Waste (Organic)**
- Food scraps, vegetable peels, fruits, leaves
- Disposal: Compost bin or organic waste facility

### 2. **Dry Waste**
- Paper, cardboard, cloth, newspaper, magazines
- Disposal: Dry waste bin for recycling centers

### 3. **Recyclable Waste**
- Plastic, glass, metal, aluminum, tin cans
- Disposal: Recyclable waste bin for material recovery

### 4. **Hazardous Waste**
- Batteries, electronics, chemicals, paint, medicines
- Disposal: Specialized collection centers and e-waste facilities
